tomblcss - my basic css framework

tomblcss is my very basic css framework. It originally started out as me making the css for this website, then, when I realised it felt like I had made the same stuff out of CSS before, it turned into a framework other people could use and I could use in other projects.
Click here to see a demo website.

Installation

You need to put the filetombl.css in the root of your web server, then add in the head of your pages. Then, go to tombl.css and edit the top bit to your likeings, editing the font-family and color stuff

Usage

You can use code or

xmp tags for showing code. code tags for one-liners or xmp tags for larger blocks of code.

You can make buttons with a tags with the class button
The code: <a class="button" href="http://example.com">Example Button</a> makes the button: Example Button
